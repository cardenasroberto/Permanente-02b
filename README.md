Procedimiento: El usuario deberá ingresar a python y utilizar los siguientes algoritmos

Algoritmo Bubble Sort:

python3 bubble_sort.py
Algortimo Insertion Sort:

python3 insertion_sort.py
n debe tener un valor por el cual el usuario tiene que cambiarlo. Para el ejemplo se usará un n igual a 100000

n = 100000 Para terminar el usuario debe ejecutar el código y el programa le mostrará el tiempo que tomó la ejecución.
